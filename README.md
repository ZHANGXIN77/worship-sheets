# 敬拜团歌谱

米兰复兴教会敬拜团专用歌谱阅读器。自动从 Dropbox 读取本周歌谱，支持离线使用。

---

## 打开方式

用 iPad 或手机 Safari 浏览器打开网址，点击底部「分享」→「添加到主屏幕」，即可像 App 一样使用。

---

## 基本操作

| 操作 | 说明 |
|---|---|
| 左右滑动 | 切换歌谱页 |
| 双指缩放 | 放大 / 缩小 |
| 双击 | 恢复原始大小 |
| 点底部曲目 | 跳转到指定歌曲 |
| 点右上角 ↻ | 手动刷新 |

---

## 离线使用

歌谱图片下载后会自动缓存到设备本地。**建议周六晚上提前打开一次**，把本周歌谱预先下载好，主日早上不依赖网络也能正常使用。

---

## 其他功能

- **敬拜流程**：歌谱页顶部可记录当天敬拜顺序，多人实时同步
- **添加链接**：可为每首歌附加参考连结（示范视频等）
- **批注涂写**：可在歌谱上临时标注，退出即清除
- **历史查阅**：主页列表可翻看过去每周的歌谱

---

## 技术说明（管理员）

- 单文件 PWA，托管于 GitHub Pages，无需服务器
- 歌谱图片存储于 Dropbox 团队空间（只读权限）
- 敬拜流程同步基于 Firebase Realtime Database
- 凭据配置见 `index.html` 顶部常量；Dropbox 完整凭据备份见 `dropbox-credentials.json`（勿提交到公开仓库）
